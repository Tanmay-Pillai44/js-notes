Functions

A function is a block of code that performs a specific task.

function declaration Vs function Expression

=> function declaration : A function declaration defines a function that will be executed when it is invoked.
Function declarations load before any code is executed . 

            function func(parameter1,parameter2){
                console.log('function declaration')
            }
                func();

=> Function Expression : A JavaScript function can also be defined using an expression.
Function expressions load only when the interpreter reaches that line of code

   A function expression can be stored in a variable:

            var func = function(parameter1,parameter2){
                console.log('function expression')
            }
            func();

Constructor Function

In JavaScript, a constructor function is used to create objects. To create an object from a constructor function, we use the new keyword.

        // constructor function
        function Person () {
            this.name = 'John',
            this.age = 23
        }

        // create an object
        const person = new Person();
