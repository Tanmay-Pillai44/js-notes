Operators in Javascript

1. Arithmetic Operators

    +, -, *, /, %, ++, --

    Ex:
    let x = 5;
    let y = 2;
    let sum = x + y;
    let sub = x - y;
    let multiply = x * y;
    let division = x / y;
    let remainder = x % y;

    ++x (pre increment), x++ (post increment)
    ++y (pre decrement), y++ (post decrement)

2. Comparison Operators

    ===, >==, <==, >==, !==
    It only returns boolean value i.e. true or false

    Ex:
    let x = 5;
    let y = 2;
    console.log(x === y); // false

3. Logical Operators

    && (and), || (or), ! (not)

    Ex:
    let x = 5;
    let y = 2;
    if (x === 5 && y === 1) {
        console.log("true");
    }
    else {
        console.log("false");
    }

4. Assignment Operators

    =, +=, -=, /=, %=

    Ex:
    let x = 5;
    x += 2;
    console.log(x); // 7

5. Ternary Operator

    condition ? exprIfTrue : exprIfFalse

    Ex:
    let x = 5;
    let ans = (x >== 3) ? "Yes" : "No";
    console.log(ans);

6. Bitwise Operators

    &   AND
    |   OR
    ^   XOR
    ~   NOT
    <<  Left Shift bitwise
    >>  Right Shift bitwise
    >>> Zero Fill Right Shift
