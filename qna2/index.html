Q 1 : Given array of students marks [23, 45, 67, 78, 91], add 10 in all the elements of an array by using map and forEach function
Ans: 
            let arr = [23, 45, 67, 78, 91]
Using map: 
            let ans = arr.map(num => num + 10);
            console.log(ans);
Using forEach: 
            let newArr = [];
            arr.forEach(num => newArr.push(num + 10));
            console.log(newArr);

Q 6 : Create object by all 4 possible way and explain with example.
Ans:

There are 4 ways to create objects in JS :

  • Object Literals
  • New operator or constructor.
  • Object.create method.
  • Object using constructor:

Object Literals:

An object literal, also called an object initializer, is a comma-separated set of paired names and values.

            var obj={
                name: "Omkar",
                age: 24,
            }

New operator or constructor:

If you call a function using a new operator, the function acts as a constructor and returns an object.

            function Car(model, color) {
                this.model = model;
                this.color = color;
            }

            var c1 = new Car('BMW', 'red');

Object.create method:

Object.create() method, which allows you to specify the prototype object and the properties.

            var Car = {
                model: 'BMW',
                color: 'red'
            }
            // You can use the Car object as a prototype to create another object.
            var ElectricCar = Object.create(Car);

Object using constructor:

            var obj=new Object();
            obj.name="Tanmay";
            obj.age="22";

Q 7 : Create an object contain name age salary as the properties, retrieve key and values in seperate arrays
Ans: 
            var obj = {
                name: "Alex",
                age: 22,
                salary: 20000
            }

            console.log(Object.keys(obj)); // [name, age, salary]

Q 8 : Explain call bind and apply in JS
Ans: 
            function greeting () {
                console.log(`Hi, I am ${this.name} and I am ${this.age} years old`);
            }
            const john = {
                name: 'John',
                age 24
            }
            const jane = {
                name: 'Jane',
                age 22
            }

call() and apply():

Both sets this keyword inside the function and immediately executes that function. Only the difference is apply() methods accpets an array of arguments instead of comma seperated values.

            function.call(thisArg, Arg1, Arg2);
            function.apply(thisArg, [agrumentsArr])

            Ex:
            greeting.call(john); // "Hi, I am John and I am 24 years old"
            greeting.apply(jane); // "Hi, I am Jane and I am 22 years old"

bind():

It creates a new function and sets the this keyword to the specified object.

            const greetingJohn = greeting.bind(john);
            greetingJohn(); // "Hi, I am John and I am 24 years old"
            OR
            greeting.bind(john)();

Q 9 : Explain Prototype Chaining
Ans:
  1. Prototypes are the means of inheritance in JS.
  2. The Prototype of ana Object could also have prototype object and this continues until we reach the top level, where no prototype object is there.

This is called Prototype Chaining.

            // pocket -> bed -> table -> head

            let head = {
                glasses: 1
            }

            let table = {
                pen: 3,
                __proto__: head
            }

            let bed = {
                sheet: 1,
                pillow: 2,
                __proto__: table
            }

            let pockets = {
                money: 2000,
                __proto__: bed
            }
